More about me:

I love Python. Python is love, Python is life (in my programming universe). This love story started in late 2011, when I for the first time saw a guy open up his laptop's webcam programmatically with OpenCV and Python. That was just... magic! Well, that's the high level abstraction of things you can usually find in almost all Python libraries. I had seen Python code before that, but really hadn't seen why I'd prefer it over C++ or Java. Had it not been for the love I found for Python in 2011, I'd probably be developing Android apps in Java or doing backend development in Java at an MNC, which is cool too, except that it wasn't my cup of tea for development if a short product POC ETA is preferred or if I need to hack out an idea as fast as possible. For me, that's the beauty of Python - the minimal time to bring out an idea to life.

In the future though, I'm planning out things to implement in Go lang, so that's an area I'd love to explore in future jobs or endeavours of mine. Currently out of the 8 projects that I've built myself for myself, 3 of which were to be businesses, are all written in Python (with the exception of one written partially in Lua and C++ too, as the components required).
